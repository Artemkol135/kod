## 📖 Описание проекта
**Training Planner** — это десктопное приложение с графическим интерфейсом (GUI), разработанное для удобного ведения дневника физических тренировок. Программа позволяет систематизировать данные о спортивной активности, сохранять историю в локальном файле и быстро находить нужные записи с помощью фильтров.

Проект демонстрирует навыки работы с:
*   Созданием интерфейсов на **PyQt6**.
*   Сериализацией данных в формат **JSON**.
*   Валидацией пользовательского ввода с помощью регулярных выражений (**Regex**).

---

## ✨ Основные возможности

### 1. Управление данными
*   **Добавление тренировок:** фиксация даты, типа активности и её длительности.
*   **Автоматическое сохранение:** база данных `trainings.json` обновляется мгновенно после каждого успешного добавления.
*   **Загрузка данных:** при старте приложение автоматически подтягивает все прошлые записи.

### 2. Строгая валидация (Контроль ошибок)
*   **Формат даты:** система принимает только формат `ДД.ММ.ГГГГ` (например, 30.04.2026).
*   **Длительность:** допускаются только положительные целые числа (минуты).
*   **Полнота данных:** невозможно добавить запись с пустыми полями.

### 3. Функции поиска и фильтрации
*   **По типу:** фильтрация таблицы по ключевым словам (например, «Силовая» или «Йога»).
*   **По дате:** быстрый поиск тренировок за определенный день.
