# 📚 Book Tracker Pro

**Автор:** Artem
**Версия:** 1.1 (Stable)

## 📋 Описание
Приложение на Python с графическим интерфейсом (PyQt6), предназначенное для ведения личной библиотеки прочитанных книг. Все данные сохраняются локально в формате JSON.

## 🛠 Функционал
- **Добавление и редактирование:** Введите существующее название книги, чтобы обновить информацию о ней.
- **Удаление:** Выберите строку в таблице и нажмите кнопку удаления.
- **Быстрое редактирование:** Двойной клик по строке таблицы переносит данные в поля ввода.
- **Фильтрация:** Мгновенный поиск по жанру и фильтр "толстых" книг (>200 страниц).
- **Безопасность данных:** Обработка ошибок записи/чтения файла (защита от повреждения JSON).
