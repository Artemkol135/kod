# 🎲 Random Quote Generator

**Автор:** Артём
**Версия:** 1.0
**Язык:** Python (PyQt6)

## 📋 Описание проекта
Приложение «Random Quote Generator» представляет собой интерактивный генератор случайных высказываний. Программа выбирает цитату из базы данных, отображает её пользователю и автоматически сохраняет историю просмотров в файл JSON.

## ✨ Функционал
1.  **Генерация:** мгновенный выбор случайной цитаты из встроенного списка.
2.  **История:** отображение списка всех ранее сгенерированных цитат.
3.  **Фильтрация:** возможность поиска по истории по имени автора или тематике цитаты.
4.  **Сохранение:** автоматический экспорт истории в `history.json` и импорт при запуске.
5.  **Интерфейс:** адаптивное окно на базе PyQt6.
